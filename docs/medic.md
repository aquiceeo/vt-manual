# Podstawy - medyczne
Lecząc kogoś w Armie warto postępować po kolei, żeby o niczym nie zapomnieć. SLuży do tego akronim SMARCHE rozwinięty niżej.

**Połataj** 
- turniquet/cat = mało czasu, rany kończyn  
- rany postrzałowe = packing bandage  
- obtarcia, stłuczenia = basic bandage  
- rany otwarte = elastic bandage  
- quick clot = rany klaty  
**Krew**  
- mało krwi = niskie ciśnienie = podaj krew  
- żółty status krwi = wstanie jeśli serce pracuje  
- pomarańczowy, czerwony status krwi = nie wstanie bez podania krwi  
**Pompuj**  
- niski puls = epipen  
- brak pulsu = epipen + CPR 
**Zaszyj**  
- szycie (dla medyka)  
- splint = złamania  
- ból = painkillers  
- duży ból = morfina (tylko przy wystarczająco wysokim tętnie)  

"Przenieś go jak paczkę", "Połatuj go jak lalkę", "Dolej mu soku", "Pompuj jak na rowerze", "Szyj mu nowy ciuch"
